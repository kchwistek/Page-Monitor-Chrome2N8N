How to Use

  1. Navigate to the page you want to monitor
  2. Open this monitoring page (right-click extension icon → Monitor Page)
  3. Enter a CSS selector for the HTML block you want to monitor
  4. Set the refresh interval (how often to check for changes)
  5. Enable/disable change detection as needed
  6. Click "Start Monitoring" to begin

Tip: Use browser DevTools (F12) to inspect elements and find the right CSS selector.

Privacy

  • Content is extracted and sent directly to your configured webhook
  • No data is stored permanently
  • Monitoring configuration is stored locally in your browser
